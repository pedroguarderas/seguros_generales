Capítulo 2 Preliminares

2.1 Notación matemática

  1. \(\mathbb{N}\) el conjunto de los números naturales \(\mathbb{N}= \{0,1,2,\ldots,\}\)

  2. \(\mathbb{N}^*\) el conjunto de los números naturales positivos \(\mathbb{N}^* = \mathbb{N}\setminus \{0\} = \{1,2,3,\ldots,\}\)

  3. \(\mathbb{Z}\) el conjunto de los números enteros, \(\mathbb{Z}= \{\ldots,-2,-1,0,1,2, \ldots \}\)

  4. \(\mathbb{R}\) el conjunto de los números reales

  5. \(\mathbb{R}_+\) el conjunto de los números reales no negativos \(\mathbb{R}_+ = \{ x \in \mathbb{R}\mid x \geq 0 \}\)

  6. \(\mathbb{C}\) el conjunto de los números complejos, \(\mathbb{C}= \mathbb{R}\oplus i \mathbb{R}\)

  7. \(\overline{\mathbb{R}} = \mathbb{R}\cup \{ +\infty \} \cup \{ -\infty \}\) el conjunto de los reales extendido incluyendo los infinitos

  8. \(x \approx y\) indica que el valor \(y\) es aproximado al valor \(x\)

  9. \(f: X \longrightarrow Y\) la función que toma valores en \(X\) y entrega valores en \(Y\)

  10. \(\sum\limits_{k=0}^n x_k\) es la suma de los elementos \(x_0,\ldots,x_n\) de una lista

  11. \([ a, b ] = \left\{ x \in \mathbb{R}\mid x \geq a \land x \leq b \right\}\) intervalo cerrado para cualquier \(a,b \in \overline{\mathbb{R}}\)

  12. \((a,b) = \left\{ x \in \mathbb{R}\mid x > a \land x < b \right\}\) intervalo abierto para cualquier \(a,b \in \overline{\mathbb{R}}\)

  13. \(\{x_n\}_{n\in\mathbb{N}} = \left\{ x_n \in X \mid n \in \mathbb{N}\right\}\) secuencia en el conjunto \(X\), no es más que una función de \(\mathbb{N}\) con valores en \(X\)

  14. \(\underset{x \rightarrow y}{\lim} f(x) = a\) límite de la función \(f(x)\) cuando \(x\) tiende a \(y\).

  15. \(\inf A\) es la mayor de las cotas inferiores de \(A\).

  16. \(\sup A\) es la menor de las cotas superiores de \(A\).

2.2.3 Estructuras básicas

Para definir una variable utilizamos el operador de asignación <-, también se puede utilizar =.

Un vector se define con la función de concatenación c

Code 
x <- c( 1, 2, 3 )
print( x )
## [1] 1 2 3

Una función se define con la sentencia function

Code 
f <- function( t ) {
  return( t^2 )
}

Si buscamos aplicar una función sobre un vector, podemos utilizar sapply

Code 
y <- sapply( x, FUN = f )
print( y )
## [1] 1 4 9

Code 
z <- lapply( x, FUN = f )
print( z )
## [[1]]
## [1] 1
## 
## [[2]]
## [1] 4
## 
## [[3]]
## [1] 9

Para muchas tareas que están relacionadas con el manejo de datos, podemos utilizar las funcionalidades del paquete de R, data.table. Para trabajar con fechas recomendamos utilizar el paquete lubridate. Para trabajar con distribuciones de probabilidad que se utilizan para modelar valores extremos utilizamos fExtremes. Para varias funcionalidades asociadas a la estimación de modelos de pérdida utilizamos actuar.
